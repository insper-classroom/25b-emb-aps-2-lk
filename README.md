- Jogo: AimLab
- Ideia do controle: simular o mouse em um protótipo de arma
- Input e Outputs:
    Botão de Liga/Desliga [branco],
    Botão de Disparo [vermelho],
    Botão de "Burst" (mais de um disparo por vez) [verde],
    Botão de Recarregar [azul],
    Led Indicando o Funcionamento
- Protocolos utilizados: GPIO, I2C, IRQ
